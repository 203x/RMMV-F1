方法

canPlayVideoType ( type ) Boolean [static]

检测浏览器能否播放指定格式的视频。

参数：

  • type String

    待检测的视频格式

返回值：

Boolean:
若浏览器可以播放该格式视频，返回 true 

canUseDifferenceBlend () Boolean [static]

检测 canvas 是否支持差值混合模式。

返回值：

Boolean:
若支持该混合模式，返回 true 

canUseSaturationBlend () Boolean [static]

检测 canvas 是否支持饱和度混合模式。

返回值：

Boolean:
若支持该混合模式，返回 true 

endLoading () [static]

清除“正在读取中”状态的图片。

hasWebGL () Boolean [static]

检测浏览器是否支持 WebGL。

返回值：

Boolean:
若浏览器支持 WebGL，返回 true 

hideFps () [static]

隐藏 FPS 显示。

initialize ( width , height , type ) [static]

初始化画面系统。

参数：

  • width Number

    游戏画面的宽度

  • height Number

    游戏画面的高度

  • type String

    渲染器的类型（'canvas'、'webgl' 或 'auto'）

isFontLoaded ( name ) Boolean [static]

检测字体文件是否载入。

参数：

  • name String

    字体的文件名

返回值：

Boolean:
若成功载入，返回 true 

isInsideCanvas ( x , y ) Boolean [static]

检测指定点是否在游戏的 canvas 区域内。

参数：

  • x Number

    指定点的 X 坐标

  • y Number

    指定点的 Y 坐标

返回值：

Boolean:
若指定点在 canvas 内，返回 true 

isVideoPlaying () Boolean [static]

检测是否正在播放视频。

返回值：

Boolean:
若正在播放视频，返回 true 

isWebGL () Boolean [static]

检测渲染器的类型是否为 WebGL 。

返回值：

Boolean:
若渲染器类型为WebGL，返回 true 

loadFont ( name , url ) [static]

读取字体文件。

参数：

  • name String

    字体文件的文件名

  • url String

    字体文件的地址

pageToCanvasX ( x ) Number [static]

将页面的 X 坐标转换为 canvas 中与之对应的 X 坐标。

参数：

  • x Number

    页面的 X 坐标

返回值：

Number:
canvas 中与之对应的 X 坐标

pageToCanvasY ( y ) Number [static]

将页面的 Y 坐标转换为 canvas 中与之对应的 Y 坐标。

参数：

  • y Number

    页面的 Y 坐标

返回值：

Number:
canvas 中与之对应的 Y 坐标

playVideo ( src ) [static]

开始播放视频。

参数：

  • src String

    视频文件的地址

printError ( name , message ) [static]

在屏幕上显示错误信息。

参数：

  • name String

    错误信息的标题

  • message String

    错误信息的内容

render ( stage ) [static]

在屏幕上渲染指定的 stage（场景） 。

参数：

  • stage Stage

    需要渲染的 stage（场景） 对象

setLoadingImage () [static]

设置“正在读取中”状态的图片。

showFps () [static]

显示 FPS 。

startLoading () [static]

初始化“正在读取中”状态的计时器。

tickEnd () [static]

帧结束的标志（用于计算 FPS ）。

tickStart () [static]

帧开始的标志（用于计算 FPS ）。

updateLoading ()  [static]

累加“正在读取中”状态的计时器时间，必要时，显示“正在读取中”状态设置的图片。
