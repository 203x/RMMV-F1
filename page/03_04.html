方法

clear () [static]

清除全部的输入数据。

initialize () [static]

初始化输入系统。

isLongPressed  ( keyName ) Boolean [static]

检查某个按键是否被长时间按下。

参数：

  • keyName String

    对应按键的映射名。

返回值：

Boolean:
若按键长按则返回 True 。

isPressed ( keyName ) Boolean [static]

检查按键是否被按下。

参数：

  • keyName String

    对应按键的映射名。

返回值：

Boolean:
若按键被按下则返回 True 。

isRepeated ( keyName ) Boolean [static]

检查按键是否被重复按下。

参数：

  • keyName String

    对应按键的映射名。

返回值：

Boolean:
若按键被重复按下则返回 True 。

isTriggered ( keyName ) Boolean [static]

检查按键是否刚被按下。

参数：

  • keyName String

    对应按键的映射名。

返回值：

Boolean:
若按键刚被按下则返回 True 。

update () [static]

更新输入数据。

属性

date Number [static]

[只读] 距离上一次按键经过的毫秒数。

dir4 Number [static]

[只读] 小键盘上对应四个方向的数字，无方向则为0。

dir8 Number [static]

[只读] 小键盘上对应八个方向的数字，无方向则为0。

gamepadMapper Object [static]

一个将手柄按键转换为按键映射名的散列表。

keyMapper Object [static]

一个将虚拟按键转换为按键映射名的散列表。

keyRepeatInterval Number [static]

The interval of the key repeat in frames.

（待议）重复按键在一帧中的持续时间。

keyRepeatWait Number [static]

The wait time of the key repeat in frames.

（待议）重复按键在一帧中的等待时间。
