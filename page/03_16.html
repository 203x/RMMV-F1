方法

clear () [static]

清除全部的输入数据。

initialize () [static]

初始化输入系统。

isCancelled () Boolean [static]

检测鼠标右键是否刚刚被按下。

返回值：

Boolean:
若鼠标右键刚刚被按下，返回 true

isLongPressed () Boolean [static]

检测鼠标左键或者触屏是否长时间被按住。

返回值：

Boolean:
若鼠标左键或者触屏长时间被按住，返回 true

isMoved () Boolean [static]

检测鼠标或手指在触屏上是否发生移动。

返回值：

Boolean:
若鼠标或手指在触屏上发生移动，返回 true

isPressed () Boolean [static]

检测鼠标或者触屏是否正被按下。

返回值：

Boolean:
若鼠标或触屏正被按下，返回 true

isReleased () Boolean [static]

检测鼠标左键是否弹起或者手指是否离开触屏。

返回值：

Boolean:
若鼠标左键弹起或者手指离开触屏，返回 true

isRepeated () Boolean [static]

检测鼠标左键或者触屏是否刚刚被按下，或者是伪重复按键是否发生。

返回值：

Boolean:
若鼠标左键或者触屏被重复按下，返回 true

isTriggered () Boolean [static]

检测鼠标左键或者触屏是否刚刚被按下。

返回值：

Boolean:
若鼠标左键或者触屏刚刚被按下，返回 true

update () [static]

更新输入数据。

属性

date Number [static]

[只读]距离上一次按键所经过的时间（毫秒）。

keyRepeatInterval Number [static]

更新重复按键状态的间隔时间（帧）。

keyRepeatWait Number [static]

进入长按状态前的等待时间（帧）。

wheelX Number [static]

[只读]水平滚动量。

wheelY Number [static]

[只读]垂直滚动量。

x Number [static]

[只读]最近一次触屏事件在 canvas 区域中的 X 坐标。

y Number [static]

[只读]最近一次触屏事件在 canvas 区域中的 Y 坐标。
