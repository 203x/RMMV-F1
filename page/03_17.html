canReadGameFiles () Boolean [static]

检查浏览器是否可以读取游戏文件夹中的文件。

返回值：

Boolean:
如果浏览器可以读取游戏文件夹中的文件，返回 true

isAndroidChrome () Boolean [static]

检查浏览器是否为安卓版的 Chrome 浏览器。

返回值：

Boolean:
如果浏览器为安卓版的 Chrome 浏览器，返回 true

isMobileDevice () Boolean [static]

检查游戏平台是否为移动设备。

返回值：

Boolean:
如果游戏平台为移动设备，返回 true

isMobileSafari () Boolean [static]

检查浏览器是否为移动版的 Safari 浏览器。

返回值：

Boolean:
如果浏览器为移动版的 Safari 浏览器，返回 true

isNwjs () Boolean [static]

检查游戏平台是否为 NW.js。

返回值：

Boolean:
如果游戏平台为 NW.js，返回 true

isOptionValid ( name ) Boolean [static]

检查选项是否在查询的字符串当中。

参数：

  • name String

    选项名称

返回值：

Boolean:
如果选项在查询的字符串当中，返回 true

rgbToCssColor  ( r , g , b ) String [static]

将 RGB 值转换成 CSS 颜色字符串。

参数：

  • r Number

    红色值，范围为(0, 255)

  • g Number

    绿色值，范围为(0, 255)

  • b Number

    蓝色值，范围为(0, 255)

返回值

String:
CSS 颜色字符串
