方法

addLoadListener ( listner )

添加一个回调函数，在位图被加载时调用该函数。

参数：

  • listner Function

    回调函数

adjustTone ( r , g , b )

改变位图色调。

参数：

  • r Number

    红色通道强度（-255～255）

  • g Number

    绿色通道强度（-255～255）

  • b Number

    蓝色通道强度（-255～255）

blt ( source , sx , sy , sw , sh , dx , dy , [dw=sw] , [dh=sh] )

将源位图给定矩形区域的数据传送到当前位图的给定矩形区域内。

参数：

  • source Bitmap

    源位图

  • sx Number

    源位图矩形区域左上角的 X 坐标

  • sy Number

    源位图矩形区域左上角的 Y 坐标

  • sw Number

    源位图矩形区域的宽度

  • sh Number

    源位图矩形区域的高度

  • dx Number

    目标区域左上角的 X 坐标

  • dy Number

    目标区域左上角的 Y 坐标

  • [dw=sw] Number [可选参数]

    目标区域的宽度

  • [dh=sh] Number [可选参数]

    目标区域的高度

blur ()

对位图执行模糊效果。

clear ()

清除位图。

clearRect ( x , y , width , height )

清除位图中指定矩形区域的内容。

参数：

  • x Number

    矩形区域左上角的 X 坐标

  • y Number

    矩形区域左上角的 Y 坐标

  • width Number

    矩形区域的宽度

  • height Number

    矩形区域的高度

drawCircle ( x , y , radius , color )

填充指定的圆形区域。

参数：

  • x Number

    圆形区域圆心的 X 坐标

  • y Number

    圆形区域圆心的 Y 坐标

  • radius Number

    圆形区域的半径

  • color String

    圆形区域的颜色（CSS格式）

drawText ( text , x , y , maxWidth , lineHeight , align )

在位图上绘制描边文字。

参数：

  • text String

    要绘制的文字

  • x Number

    文字左侧的 X 坐标

  • y Number

    文字顶部的 Y 坐标

  • maxWidth Number

    文字允许的最大宽度

  • lineHeight Number

    文字的行高

  • align String

    文字的对齐方式

fillAll ( color )

填充整个位图

参数：

  • color String

    填充的颜色（CSS格式）

fillRect ( x , y , width , height , color )

填充指定的矩形区域。

参数：

  • x Number

    矩形区域的 X 坐标

  • y Number

    矩形区域的 Y 坐标

  • width Number

    矩形区域的宽度

  • height Number

    矩形区域的高度

  • color String

    填充的颜色（CSS格式）

getAlphaPixel ( x , y ) String

返回指定像素点的不透明度（alpha 值）。

Parameters:

  • x Number

    位图中指定点的 X 坐标

  • y Number

    位图中指定点的 Y 坐标

返回值：

String:
不透明度（alpha 值）

getPixel ( x , y ) String

返回指定像素点的颜色。

参数：

  • x Number

    位图中指定点的 X 坐标

  • y Number

    位图中指定点的 Y 坐标

返回值：

String:
颜色（十六进制格式）

gradientFillRect ( x , y , width , height , color1 , color2 , vertical )

以渐变模式填充指定的矩形区域。

参数：

  • x Number

    矩形区域的 X 坐标

  • y Number

    矩形区域的 Y 坐标

  • width Number

    矩形区域的宽度

  • height Number

    矩形区域的高度

  • color1 String

    渐变的起始颜色

  • color2 String

    渐变的结束颜色

  • vertical Boolean

    是否为纵向渐变

isError () Boolean

检测位图加载时是否发生错误。

返回值：

Boolean:
若发生错误，返回 true

isReady () Boolean

检测位图能否开始渲染。

返回值：

Boolean:
若位图能够开始渲染，返回 true

load ( url ) [static]

加载图片文件，返回一个新的位图对象。

参数：

  • url String

    加载图片的地址

返回值：

Bitmap（位图） 对象

measureTextWidth ( text ) Number

测量指定文字的宽度。

参数：

  • text String

    待测量的文字

返回值：

Number:
文字的宽度（像素）

resize ( width , height )

调整位图大小。

参数：

  • width Number

    位图调整后的宽度

  • height Number

    位图调整后的高度

rotateHue ( offset )

变换位图的色相。

参数：

  • offset Number

    色相环的偏移量（360度以内）

snap ( stage ) [static]

截取当前游戏画面，返回一个新的 Bitmap（位图） 对象。

参数：

  • stage PIXI.Stage

    stage（场景） 对象

返回值：

Bitmap（位图） 对象
