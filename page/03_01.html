方法

addLoadListener ( listner )

添加一个回调函数，在位图被加载的时候调用该函数。

参数：

  • listner Function

    回调函数。

adjustTone ( r , g , b )

改变位图色调。

参数：

  • r Number

    红色值（-255～255）。

  • g Number

    绿色值（-255～255）。

  • b Number

    蓝色值（-255～255）。

blt ( source , sx , sy , sw , sh , dx , dy , [dw=sw] , [dh=sh] )

将位图 source 的矩形（sx, sy, sw, sh）中数据，整体传送到当前位图的坐标 (dx, dy) 处。
dw,dh为传送之后矩形的宽和高。

参数：

  • source Bitmap

    要绘制的位图

  • sx Number

    位图source的矩形的x做标。

  • sy Number

    位图source的矩形的x做标。

  • sw Number

    位图source的矩形的宽度。

  • sh Number

    位图source的矩形的高度。

  • dx Number

    目标矩形的x坐标。

  • dy Number

    目标矩形的y坐标。

  • [dw=sw] Number [optional]

    目标矩形的宽度。

  • [dh=sh] Number [optional]

    目标矩形的高度。

blur ()

对位图执行模糊效果。

clear ()

清除位图。

clearRect ( x , y , width , height )

清除位图区域 (x, y, width, height) 。

参数：

  • x Number

    被清除区域左上角的x坐标。

  • y Number

    被清除区域左上角的y坐标。

  • width Number

    被清除区域宽度。

  • height Number

    被清除区域的高度。

drawCircle ( x , y , radius , color )

画圆。

参数：

  • x Number

    圆心的x坐标。

  • y Number

    圆心的y坐标。

  • radius Number

    圆的半径。

  • color String

    圆的颜色代码（CSS格式）。

drawText ( text , x , y , maxWidth , lineHeight , align )

绘制描边文字。

参数：

  • text String

    要绘制的文字。

  • x Number

    描绘位置的x坐标。

  • y Number

    描绘位置的y坐标。

  • maxWidth Number

    文字允许的最大宽度。

  • lineHeight

    文本行高。

  • align String

    文字的对齐方式。

fillAll ( color )

以指定颜色填充整个位图。

参数：

  • color String

    要填充的颜色（CSS格式）。

fillRect ( x , y , width , height , color )

以指定颜色填充矩形区域。

参数：

  • x Number

    要填充的矩形的x坐标。

  • y Number

    要填充的矩形的y坐标。

  • width Number

    要填充的矩形的宽度。

  • height Number

    要填充的矩形的高度。

  • color String

    要填充的颜色（CSS格式）。

getAlphaPixel ( x , y ) String

返回指定像素点的不透明度（alpha值）。

Parameters:

  • x Number

    位图中指定点的x坐标。

  • y Number

    位图中指定点的y坐标。

返回值：

String:
像素点的透明度（alpha值）。

getPixel ( x , y ) String

返回指定像素点的颜色。

Parameters:

  • x Number

    位图中指定点的x坐标。

  • y Number

    位图中指定点的y坐标。

返回值：

String:
像素点颜色(十六进制格式)。

gradientFillRect ( x , y , width , height , color1 , color2 , vertical )

以渐变模式填充矩形区域。

参数：

  • x Number

    要填充的矩形的x坐标。

  • y Number

    要填充的矩形的y坐标。

  • width Number

    要填充的矩形的宽度。

  • height Number

    要填充的矩形的高度。

  • color1 String

    渐变的开始颜色。

  • color2 String

    渐变的结束颜色。

  • vertical Boolean

    是否为纵向渐变。

isError () Boolean

检查是否出现加载错误。

返回值：

Boolean:
加载错误返回真。

isReady () Boolean

检查是否是准备渲染位图。

返回值：

Boolean:
准备渲染返回真。

load ( url ) [static]

加载一个图片并返回对应的位图对象。

参数：

  • url String

    被加载图片的地址（url）。

返回值：

Bitmap

measureTextWidth ( text ) Number

返回欲绘制文字的宽度。

参数：

  • text String

    欲绘制文字。

返回值：

Number:
欲绘制文字的宽度。

resize ( width , height )

调整位图大小。

参数：

  • width Number

    位图调整后的宽度。

  • height Number

    位图调整后的高度。

rotateHue ( offset )

变换位图的色相。

参数：

  • offset Number

    改变的色相值（360度以内）。

snap ( stage ) [static]

将游戏屏幕快照返回为一个新的 Bitmap 对象。

参数：

  • stage PIXI.Stage

    stage对象。

返回值：

Bitmap
