以平铺方式显示图片的 sprite（精灵） 对象。

构造函数

TilingSprite ( bitmap )

参数：

  • bitmap Bitmap

    需要平铺显示的图片

方法

move ( x , y , width , height )

直接设置 TilingSprite（平铺精灵） 的 X 坐标、Y 坐标、宽度和高度。

参数：

  • x Number

    TilingSprite（平铺精灵） 的 X 坐标

  • y Number

    TilingSprite（平铺精灵） 的 Y 坐标

  • width Number

    TilingSprite（平铺精灵） 的宽度

  • height Number

    TilingSprite（平铺精灵） 的高度

setFrame ( x , y , width , height )

设置 TilingSprite（平铺精灵） 所显示 bitmap（位图） 的矩形区域。

参数：

  • x Number

    矩形区域的 X 坐标

  • y Number

    矩形区域的 Y 坐标

  • width Number

    矩形区域的宽度

  • height Number

    矩形区域的高度

update ()

在每一帧中刷新 TilingSprite（平铺精灵） 。

属性

bitmap Bitmap

TilingSprite（平铺精灵） 所显示的图片。

opacity Number

TilingSprite（平铺精灵） 的不透明度（0 ~ 255）。

origin Point

滚动显示效果中 TilingSprite（平铺精灵） 的原点。

visible Boolean

TilingSprite（平铺精灵） 的可见性。

x Number

TilingSprite（平铺精灵） 的 X 坐标。

y Number

TilingSprite（平铺精灵） 的 Y 坐标。
