以平铺方式显示图片的精灵对象。

构造函数

TilingSprite ( bitmap )

参数：

  • bitmap Bitmap

    需要平铺显示的图片

方法

move ( x , y , width , height )

统一设置 x 坐标，y 坐标，宽度，高度。

参数：

  • x Number

    平铺精灵的 x 坐标

  • y Number

    平铺精灵的 y 坐标

  • width Number

    平铺精灵的宽度

  • height Number

    平铺精灵的高度

setFrame ( x , y , width , height )

指定平铺精灵所显示位图的矩形区域。

参数：

  • x Number

    矩形框架的 x 坐标

  • y Number

    矩形框架的 y 坐标

  • width Number

    矩形框架的宽度

  • height Number

    矩形框架的高度

update ()

在每一帧刷新精灵。

属性

bitmap Bitmap

用于平铺显示的图像。

opacity Number

平铺精灵的不透明度(0 ~ 255)。

origin Point

平铺精灵的原点，用于制作滚动显示效果。

visible Boolean

平铺精灵的可见性。

x Number

平铺精灵的 x 坐标。

y Number

平铺精灵的 y 坐标。
