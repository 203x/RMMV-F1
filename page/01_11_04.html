构建游戏发行包

您可以为游戏构建包括 Windows、Mac OS X、安卓和 iOS 在内全部平台的发行包。

Windows平台
选择打包后将导出游戏的工程文件夹。
如果您希望将文件夹打包成单独的 .exe 文件，请参考 [使用 Enigma Virtual Box] 部分。
Mac OS X
请发布游戏的工程文件夹以及文件夹下的 Game.app 文件。
* 在 Windows 平台下 Game.app 会显示成了一个文件夹，请不用担心，它可以在 Mac OS X 下正常运行。
Android/iOS
请参考 [转换为安卓应用] 和 [转换为 iOS 应用] 部分。
网页浏览器（March 1st, 2016 Update）
将你所导出的工程文件夹上传到服务器后，你就可以在网页浏览器中运行你的游戏了。

运行游戏所需的系统要求
系统要求 需要支持 HTML5 Canvas 以及 Web Audio API
iOS 8 或更高版本
Android 4.4 或更高版本
推荐运行环境 Windows, Android: Google Chrome 浏览器
Mac OS X, iOS: Safari 浏览器
* 在线进行游戏时请使用 Wi-Fi 。
* 推荐使用支持 WebGL 的浏览器进行游戏。

[Exclude unused files（除去未使用文件）]选项 （2016.03.01 更新）

在发布窗口中选择此选项可以在游戏发布时去掉所有不需要的文件。使用这个功能可以减小发布文件包的体积。

以下文件夹中未使用的文件将被除去。

  • 工程文件夹/img/(此文件夹中多余的文件)
  • 工程文件夹/audio/(此文件夹中多余的文件)

请参考[[Exclude unused files（除去未使用文件）] 插件设置]中设置此插件的详细说明。
